#### Installation

1. Clone the Git Repository
```
git clone https://github.com/isalew/mean-workout.git
```
2. Install package dependencies
```
cd <project-folder>
npm install
```
3. Start Mongo
```
mongod
```
4. Run server ( Opens on localhost:3000 )
```
npm start
```
